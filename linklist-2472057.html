    Pengertian Semantic & Non-semantic Elements


    Semantic Elements
    Semantic elemnts adalah elemen HTML yang memiliki arti atau makna yang jelas bagi manusia maupun browser
    Dengan hanya melihat tag HTML-nya saja, kita bisa mengetahui fungsi elemen tersebut dalam sebuah halamn web

    Non-semantic Elements
    Non-semantic elemnts adalah elemen HTML yagn tidak memiliki makna khusus dan hanya berfungsi sebagai waah untuk elemen lain
    Browser tidak dapat memahami tujuan dari elemen ini tanpa adanya informasi tambahan seperti calss atau id

    2. < table >

    4. < nav >

    7. < aside >
    8. < footer >
    9. .etc.
